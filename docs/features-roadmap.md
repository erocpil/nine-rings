# Nine Rings — 功能统计与路线图

> 生成日期：2026-07-09

---

## 已完成功能（40+ 项）

### 笔记系统
- 按天组织笔记
- 富文本编辑（TipTap / Quill Delta）
- 笔记 CRUD（创建/读取/更新/删除 + 软删除）
- 笔记置顶
- 笔记拖拽排序
- 跨日移动
- 版本历史（自动快照，最多保留 30 版，可回滚）
- 回收站（恢复 + 永久删除 + 自动清理过期）
- 键盘快捷键（Ctrl+N 新建, Ctrl+E 搜索, Ctrl+, 设置）

### 待办系统
- 添加/勾选/删除（带撤销）
- 拖拽排序（▲▼ 按钮）
- 双击编辑
- 过期待办提示（自动扫描未完成的历史待办）
- 添加待办后自动保存

### 标签系统
- 笔记标签输入、存储
- 侧栏标签筛选（点击标签筛选当前日期笔记）
- 标签支持多标签

### UI 布局
- 三栏布局（侧栏笔记列表 + 待办区 + 笔记编辑器）
- 左右分隔条可拖拽（120–500px）
- 上下分隔条可拖拽（待办/编辑器比例）
- 响应式 flex 布局
- 多窗口支持（Tauri 桌面端）

### 主题
- 系统主题、浅色、深色
- 粋（护眼绿）、雅（淡紫）、幟（鹅黄）

### 搜索
- 全文搜索（搜索笔记标题 + 正文）
- 搜索结果展示

### Header
- 时钟显示（24h 格式，每 30 秒更新）
- 日期选择器（📅 emoji 触发系统日历）
- 每日概览（周几 月/日 · 笔记数 · 待办进度）
- 搜索框（右侧紧凑布局）

### 配置系统
- 设置面板（主题、字号、默认视图、待办继承开关、自动清理天数）
- 持久化到 localStorage / SQLite

### 数据
- IndexedDB 存储（Web 端）
- SQLite 存储（Tauri 桌面端，Rust 后端）
- 抽象 StorageAdapter 层，前后端共享数据模型
- JSON Schema 定义（`schema/` 目录）
- Quill Delta 格式，带 Delta ↔ Markdown 转换层

### 导出/导入
- API 层：全量导出 JSON、导入 JSON、单篇 Markdown 导出
- UI：无（仅 API，待添加）
